# Real-Time Chat Application

A full-stack real-time chat application that allows multiple users to communicate instantly through a modern web interface.  
The project consists of a React-based frontend and a backend server responsible for handling real-time communication between connected clients.

This project demonstrates modern web development concepts including real-time messaging, client-server communication, and scalable application structure.

---

## Features

- Real-time messaging between users
- Modern React-based user interface
- Client-server architecture
- Persistent connection for live communication
- Modular frontend and backend structure
- Scalable project layout for future features

## Installation

### 1. Clone the repository

```
git clone https://github.com/ShivendraNirupam/chat-app.git
cd chat-app
```

---

## Install Dependencies

Install frontend dependencies:

```
cd frontend
npm install
```

Install backend dependencies:

```
cd ../backend
npm install
```

---

## Running the Application

Start the backend server:

```
cd backend
npm start
```

Start the React frontend:

```
cd frontend
npm start
```

The application will run locally and can be accessed through the browser.
